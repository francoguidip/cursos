# EII 147 — Introducción a las Tecnologías de Información

Sube aquí tus archivos (PDF, PPTX, etc.) para este curso.

## Cómo agregarlos al sitio

1. Sube el archivo a esta carpeta (`eii147/`) desde GitHub ("Add file → Upload files").
2. Abre `index.html` en la raíz del repositorio y edita.
3. Dentro de la lista `<ul class="materials">` del bloque de EII 147, agrega una línea nueva:

```html
<li><a href="eii147/NOMBRE-DEL-ARCHIVO.pdf"><span>Clase 1 — Introducción</span><span class="tag">pdf</span></a></li>
```

4. Reemplaza `NOMBRE-DEL-ARCHIVO.pdf` por el nombre real del archivo subido, y el texto entre `<span>` por el título que quieras mostrar.
5. Guarda los cambios ("Commit changes"). El sitio se actualiza solo en unos minutos.

Puedes borrar esta línea de ejemplo del `index.html` cuando ya no la necesites como referencia.
